# Repositório SQL com exercícios básicos de T-SQL (Microsoft)

### Motivação
Após voltar a fazer uma nova graduação dessa vez em Ciência da Computação, vi a necessidade de praticar SQL (bastante) por sinal, não ficar preso a teoria ou a exemplos que eu mesmo crio com poucas tabelas e além de poucas tabelas, poucas colunas e pouco povoadas.
Por isso consta nesse repositório, exercícios de natureza fácil/intermediária que será atualizada periodicamente com novos exercicios para fixação do aprendizado.  
Falando um português mais direto: Eu só estudei na faculdade/provas de 2 a 3 tabelas, onde cada tabela tinha 4 a 5 colunas e eram pouco povoadas (poucas linhas de dados), beleza?! 

### Banco de dados - AdventureWorksLT2025

<br>
Adventure WorksLT 2025 - [Microsoft](https://learn.microsoft.com/en-us/sql/samples/adventureworks-install-configure?view=sql-server-ver17&tabs=ssms "Melhor banco de dados para iniciante").

### Fonte dos exercícios
A fonte utilizada foi uma lista gerada pela IA Gemini, o prompt em questão foi o seguinte: "gemini, gere 100 exercícios divididos em filtros, joins, manipulação de string, operações matemáticas e datas, utilizando como exemplo o AdventureWorksLT2025". Vale ressaltar que exercícios que envolva datas por exemplo, ainda serão resolvidas.

> [!IMPORTANT]
> Utilize a IA sendo sincero a sua pessoa, não a utilize na primeira dificuldade, destine minutos e "estresse sua mente" no bom sentido, para forçar sua mente a aprender, um dos exemplos foi utilizar a IA para gerar exercícios para a prática.

#### Assuntos abordados nos exercícios
- Filtros e lógica de seleção
- Joins e relacionamentos
- Manipulação de Strings
- Operações matemáticas e financeiras
- Operações com datas
